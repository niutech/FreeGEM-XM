This directory contains some binaries specific to GEM/XM.

They may be of use when configuring FreeGEM/XM for your system.

GDOSINST.EXE will set up the RAMDrive and hotkey settings, if the
defaults (highest drive letter, keypad '+') aren't suitable.

FILTER.APP sets up keyboard filter maps.  Not sure how these work.
